# WeatherApp_POE_Complete (skeleton)

This repository is a skeleton Android project prepared for the POE submission. It includes:
- Login & Register screens (Kotlin, ViewBinding)
- SHA-256 password hashing example
- Room database skeleton to store users (offline cache)
- Retrofit scaffold to connect to a REST API
- Firebase placeholder dependencies (add google-services.json and apply plugin)
- Google Sign-In dependency stub
- Multi-language support: English + isiZulu (values-zu)
- README, AI usage write-up and video placeholder

## What you must do after downloading
1. Open the project in Android Studio.
2. Add `google-services.json` in `app/` if you want Firebase features.
3. Update `ApiService.BASE_URL` with your REST API endpoint.
4. Install required SDKs (compileSdk 34).
5. Build and run on a physical Android device.
